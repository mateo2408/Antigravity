# EcuStickers Mobile

A website showcasing Ecuadorian stickers with identity - featuring fauna, flora, and culture from Ecuador.

## Live Demo

This site is deployed using GitHub Pages. Once configured, it will be available at:
`https://mateo2408.github.io/Antigravity/`

## GitHub Pages Setup

This repository is configured for GitHub Pages deployment:

1. The website files (`index.html`, `styles.css`, `script.js`) are located in the root directory
2. GitHub Pages is configured to serve from the `main` branch, root directory (`/`)

### How to Enable GitHub Pages

If GitHub Pages is not yet enabled, follow these steps:

1. Go to the repository Settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Build and deployment":
   - Source: Deploy from a branch
   - Branch: Select `main` and `/ (root)`
4. Click "Save"

GitHub will automatically deploy the site within a few minutes.
